### &#x20;   ❤️ Heart Disease Prediction using Machine Learning

#### 

#### 

#### &#x20;📌 Project Overview



Heart disease is one of the leading causes of death worldwide. Early prediction can help healthcare professionals identify high-risk patients and support timely medical intervention.



This project develops a Machine Learning classification model to predict whether a patient is likely to have heart disease based on various clinical features.



The project follows a complete machine learning workflow including data preprocessing, exploratory data analysis (EDA), feature scaling, model training, evaluation, and comparison of multiple classification algorithms.



#### 

#### &#x20;🎯 Objectives



\- Perform Exploratory Data Analysis (EDA)

\- Clean and preprocess the dataset

\- Apply Feature Scaling

\- Train multiple Machine Learning models

\- Evaluate model performance using different metrics

\- Compare models and identify the best-performing algorithm



#### &#x20;📂 Dataset



The dataset contains patient health records with several medical attributes used to predict the presence of heart disease.

##### 

##### &#x20;Features



\- Age

\- Sex

\- Chest Pain Type

\- Resting Blood Pressure

\- Cholesterol

\- Fasting Blood Sugar

\- Resting ECG

\- Maximum Heart Rate

\- Exercise Induced Angina

\- ST Depression

\- Slope

\- Number of Major Vessels

\- Thalassemia



\### Target Variable



\- 0 → No Heart Disease

\- 1 → Heart Disease

#### 

#### &#x20;🏆 Results



After comparing all three models, Random Forest achieved the best overall performance with the highest predictive accuracy and balanced classification metrics.



Random Forest was selected as the final model due to its robustness and superior performance on the dataset.
